
Cedar Havana - PWA package

Files included:
- index.html
- manifest.json
- service-worker.js
- icons/ (app icons and splash)

How to host and install (Netlify recommended):
1. Unzip the Cedar_Havana.zip to a folder.
2. Go to https://app.netlify.com/drop and drag the extracted folder to deploy.
3. After deployment, open the published URL in Safari on your iPhone.
4. Tap Share -> Add to Home Screen. The app will install as "Cedar Havana".
5. Once installed it works offline.

If you prefer GitHub Pages or another host, upload the folder contents to the server root.

Notes:
- To update the app later, replace the files and redeploy.
- The PWA uses a simple service worker for offline caching.
